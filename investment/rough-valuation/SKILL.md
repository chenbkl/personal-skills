# 毛估估估值法（私有化退市 / 目测法）

> **方法来源**：本 skill 复刻自 **张局营（雪球 ID「闲来一坐s话投资」）** 的「毛估估」估值方法。茅台 / 片仔癀 / 格力三例均出自其公开文章。第 5 节的「Claude 校准层」为后加补充，非原作者方法。

把这套「毛估估」估值套路固化成可复用流程。核心思想：**假设把整家公司"私有化退市"自己持有，10 年里它能给你掏出多少真金白银，划不划算？** 用算术级的口算，得出"模糊的正确"，而不是精确的错误。

参考案例见同目录 `examples.md`（茅台 / 片仔癀 / 格力，含已核对的数字）。

## 2. 适用前提（先过这道闸，不过就不能用毛估估）

作者反复强调：这套方法**只适用于轻资产、赚"真"钱、有护城河的公司，不具普遍性**。开算前先核对：

| 闸门 | 怎么判断 | 不通过的后果 |
|---|---|---|
| **轻资产，净利润≈自由现金流** | 看「经营活动现金流量净额」是否长期 ≥ 净利润；资本开支小 | 重资产/靠垫资的公司，净利润是"假"钱，累计利润法严重失真 |
| **赚"真"钱（主业造血）** | 利润主要来自核心利润，不是投资收益/政府补助/一次性处置/减值转回 | 利润不可持续，增长假设无意义 |
| **有护城河** | 定价权、品牌商誉，能支撑"未来 10 年增长"的假设 | 增长率纯属幻想 |
| **能力圈内、吃透基本面** | 用户是否真的理解这门生意 | 不可盲目照搬 |

**反例（直接警告用户"此方法不适用，算了也不可信"）**：银行/保险/券商、重资产制造与周期股、净利润远大于经营现金流的公司、靠减值转回或投资收益撑利润的公司。

> 茅台 / 片仔癀 / 格力是作者明确认可的适用对象（净利润可近似当自由现金流）。

## 3. 计算步骤

记：P0 = 最新年度扣非净利润，MC = 当前市值，g = 假设的年复合增长率。

### Step 0 — 静态收益率（"利润固定不增长"的底线）
```
静态收益率 = P0 / MC = 1 / 静态PE
```
先报这个数。它是"假设利润永远停在今天"的回报率（片仔癀 2.25%、格力 12.71%）。和后面带增长的对比，看增长贡献多少。

### Step 1 — 增长情景（线性外推，做三档）
默认做 **10% / 15% / 20% 三档**（作者通常算到 15%，20% 多数公司"不敢奢望"）。先用近 5–10 年净利润 CAGR 验证哪档现实。
```
第10年利润  = P0 × (1+g)^10
第10年静态收益率 = 第10年利润 / MC
```
（1.10^10≈2.594，1.15^10≈4.046，1.20^10≈6.192）

### Step 2 — 累计净利润法（私有化退市核心）
```
10年累计利润 = Σ P0×(1+g)^n,  n=1..10
剩余成本     = MC − 10年累计利润
第10年回报率 = 第10年利润 / 剩余成本
```
逐年列出每年利润再相加。语义："私有化后把这些利润全花掉，剩余成本只够买一家'还在年赚第10年利润'的公司，这门生意自家有划不划算"。
（等比求和速算：Σ(1+g)^1..10 = (1+g)×((1+g)^10 − 1)/g；10%→17.53，15%→23.35，20%→31.15）

### Step 3 — 分红法（高分红公司用，如茅台/格力）
```
某年每股分红 = 当年EPS × 分红率         (EPS = 当年净利润/总股本)
10年累计每股分红 = Σ 各年每股分红
持股成本下降到 = 买入价 − 10年累计每股分红
```
语义："一鸟在手胜过两鸟在林"，只靠分红 10 年能收回多少本。注意分红率若公司公告会提升（如茅台 51.9%→75%），用新分红率。

### Step 4 — 机会成本对比（估值锚）
```
国债等价PE = 100 / 10年期国债收益率(%)
```
对比：公司**静态PE** vs 国债等价PE；公司**盈利收益率(1/PE)** vs 国债收益率。
话术："债券是特殊的企业，企业是特殊的债券。" 一个不增长的国债市场愿给 N 倍，一个还在增长的好公司只要 M 倍（M<N），就是机会成本意义上的便宜。

### Step 5 — 目测法（低增长 / 低估值简化版）
增长乏力或只想快速判断时，跳过 Step 1–3，直接：
```
静态收益率(1/PE) + 股息率
```
看：这么低的 PE 是不是已经price in了"永不增长"的悲观预期？"自家有这么个生意，满不满意？"（格力案例：静态收益率 12.71% + 股息率 5.88%）。

## 5. ⚠️ Claude 校准层（固定追加，与作者方法分隔清楚）

> 以下为 Claude 补充，**非作者原方法**。作者主动选择不折现、不纠结估值波动；这里把被省略的另一面补上，避免被面值大数字误导。

1. **折现提醒**：Step 2 的"累计净利润"是**面值、未折现**，会系统性高估便宜程度（越往后的利润越被高估）。速算：当折现率≈增长率时，每年利润折现回今天≈P0，10 年现值≈10×P0（而非面值累计的十几倍）；"约每 7.2/r 年价值减半"。低增长公司尤其要打折看。
2. **买入 PE / 戴维斯双杀**：增长和分红是"生意"，但**你买入时的 PE 才决定结局**——这是唯一你能 100% 控制的变量。报出当前 PE 在历史 1/3/5/10 年分位（理杏仁已给）。反例提醒：茅台 2021 年 70 倍买入，利润 3 年涨 60%，股价却跌 44%，因 PE 杀到 25 倍（戴维斯双杀）。
3. **总回报拆解**：
   ```
   长期年化总回报 ≈ 利润增速 + 股息率 ±（卖出PE ÷ 买入PE 的变化）
                    └──生意，可估──┘   └──市场情绪，看运气──┘
   ```
4. **核心风险**：整套结论压在"未来 10 年增长率"这一个假设上。用近年**实际**增速回测假设是否落空，并指出这是需要持续盯的核心变量。

## 6. 重要声明（复刻作者原话，输出末尾保留）

- 以上**未考虑货币时间价值（未折现）**——这是作者的主动选择，他认为一考虑折现率就把简单问题复杂化了。（反面见第 5 节）
- 仅依据历史数据做**线性外推**，是"思想实验 / 毛估估"，不是精确预测；实际各年增速有快有慢。
- **模糊的正确 > 精确的错误。**
- 仅提供"解题（估值）思路"，**不是买卖信号**；必须建立在吃透公司基本面、在自己能力圈之内，**不可盲目照抄照搬**。
